Prog    -> Dcls Stmts $

Dcls    -> Dcl Dcls
         | å…¥

Dcl     -> floatdcl id
         | intdcl id

Stmts   -> Stmt Stmts
         | å…¥

Stmt    -> id assign Exprh
         | print id

Exprh   -> Exprl
         | Exprl Opeh Exprh

Exprl   -> Val
         | Val Opel Exprl

Opeh    -> plus
         | minus

Opel    -> mult
         | div

Val     -> id
         | inum
         | fnum
