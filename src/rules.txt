Prog	-> Dcls Stmts $
Dcls	-> Dcl Dcls
		 | å…¥
Dcl		-> floatdcl id
		 | intdcl id
Stmts	-> Stmt Stmts
		 | å…¥
Stmt	-> id assign Val Exprh
		 | print id
Exprh	-> plus Val Exprh
		 | minus Val Exprh
		 | Exprl
		 | å…¥
Exprl	-> mult Val Exprl
		 | div Val Exprl
		 | å…¥
Val		-> id
		 | inum
		 | fnum
