Prog    -> Dcls Stmts $

Dcls    -> Dcl Dcls
         | å…¥

Dcl     -> floatdcl id
         | intdcl id

Stmts   -> Stmt Stmts
         | å…¥

Stmt    -> id assign Exprh
         | print id

Exprh   -> Exprl Opeh Exprh
         | Exprl

Exprl   -> Val Opel Exprl
         | Val

Opeh    -> plus
         | minus

Opel    -> mult
         | div

Val     -> id
         | inum
         | fnum
