Prog	-> Dcls Stmts $
Dcls	-> Dcl Dcls
		 | å…¥
Dcl		-> floatdcl id
		 | intdcl id
Stmts	-> Stmt Stmts
		 | å…¥
Stmt	-> id assign Exprh
		 | print id
Exprh	-> Exprh Opeh Exprh
		 | Exprl
		 | å…¥
Exprl	-> Exprl Opel Exprl
		 | Val
		 | å…¥
Opeh	-> plus
		 | minus
Opel	-> mult
		 | div
Val		-> id
		 | inum
		 | fnum
